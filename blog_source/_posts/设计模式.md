# 设计模式

设计模式的相关复习。 先把我用过的设计模式梳理一下

## 单例模式

很常见

Java 实现单例的方法有很多

JavaScript呢？

闭包实现单例

``` JavaScript

function A () {
  this.data = "Class A"
}

A.getInstance = function () {
  var instance;
  return function () {
    instance = instance ? instance: new A
    return instance
  }
}()
```

正常单例

``` JavaScript
let A = function () {
  if (A.instance) {
    console.log("no new")
  }
  this.data = "Class A"
}

A.getInstance = function () {
  if (A.instance == null) {
    A.instance = new A()
  }
  return A.instance
}

```
