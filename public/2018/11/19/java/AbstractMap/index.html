空的构造方法，

size

isEmpty

调用size(), 看看size是否为零

containsKey

调用entrySet() 获取Set,调用Set.iterator()获取一个迭代器，使用这个迭代器进行遍历查询
需要注意的是,在方法内部使用了if语句，通过Object key是否为空来确定使用　== null 还是　equals 来进行判断是否匹配
